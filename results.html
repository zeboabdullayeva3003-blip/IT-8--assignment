A cloud-hosted web application built for the Introduction to IT course assignment.

Project Overview

EduFeedback is a course feedback web application that demonstrates core cloud computing concepts including deployment, storage, and web hosting using a Platform-as-a-Service (PaaS) model.

Technology Stack

  • Python Programming language
  • Flask Web framework
  • HTML/CSS Frontend interface
  • JSON Data storage
  • GitHub Version control
  • Render.com Cloud deployment (PaaS)

Features

  • ✓ HTML input form with validation
  • ✓ Server-side data processing
  • ✓ Persistent data storage
  • ✓ Results dashboard with statistics
  • ✓ Multi-page navigation
  • ✓ Responsive CSS styling
  • ✓ Deployed on cloud (Render.com)
